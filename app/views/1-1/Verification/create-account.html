Create a DfE Sign-in account

A DfE Sign-in account will allow you to access DfE online services. An approver at your organisation will need to create an account for you.

What is an approver?

An approver can create user accounts, add and remove services and update roles within organisations.

Find an approver

Approver contact details can be provided by:

  • the headteacher
  • the administration team
  • the account manager
  • the apprenticeship manager

There are no approvers at my organisation

To set up the first approver at your organisation, you will need to send an email to the Service Team.
